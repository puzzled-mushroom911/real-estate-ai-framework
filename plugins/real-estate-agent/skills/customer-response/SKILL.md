# Customer Response

Draft a ready-to-send customer response that matches the agent's communication style and is backed by accurate local knowledge.

## Process

1. **Classify the inquiry:**
   - Buyer (hot/warm/cold), seller, general question, objection, or referral
   - Determine response priority and appropriate length

2. **Draft the response:**
   - Match the agent's tone, vocabulary, greeting style, and sign-off
   - Answer the question directly with specific data
   - Include one insight the customer didn't ask for but will value
   - End with an appropriate CTA (soft for cold leads, direct for hot)
   - Keep it proportional to the question length

3. **Verify before delivering:**
   - All facts accurate and verifiable
   - Fair Housing compliant (use "buyers" not "families", no steering)
   - No generic realtor language ("dream home", "turnkey", etc.)
   - Labeled as DRAFT -- never auto-sent

## Output Format

```
## Response Draft
**To:** [Contact name]
**Type:** [Inquiry classification]
**Priority:** [Immediate / Same Day / 24 Hours]

---
[Ready-to-send message]
---

**Notes:** [Any caveats or items needing the agent's input]
```
